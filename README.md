# PSX Trading Calculator Pro v1.0.0

Professional desktop application for PSX trading calculations.

## Features
- **Position Size Calculator** - Calculate exact shares to buy
- **Stop Loss Manager** - 3 methods (Manual, Percentage, ATR)
- **Risk/Reward Calculator** - Auto targets (1:1, 1:2, 1:3)
- **Brokerage Calculator** - PSX charges included
- **Trade Journal** - Track all trades with statistics
- **Export Options** - Excel, CSV, PDF formats
